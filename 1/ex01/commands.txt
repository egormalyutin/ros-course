===
Название: fdisk
Описание: Утилита для создания и управления таблицами разделов на диске. Флаг -l используется для просмотра списка разделов на всех дисках без внесения изменений.
Синтаксис: sudo fdisk -l [УСТРОЙСТВО]
Пример: sudo fdisk -l /dev/sda
Примечание: Запуск команды без флага -l переведет ее в интерактивный режим, где можно случайно удалить разделы.

===
Название: ncdu (NCurses Disk Usage)
Описание: Интерактивная утилита для анализа использования дискового пространства. Аналог du, но с простым текстовым интерфейсом, где можно легко navigровать по каталогам и удалять файлы.
Синтаксис: ncdu [ОПЦИИ] [КАТАЛОГ]
Пример: sudo ncdu /
Примечание: Очень безопасна для навигации. Удаление файлов происходит только после подтверждения.

===
Название: rsync
Описание: Мощная утилита для копирования и синхронизации файлов и каталогов как локально, так и по сети. Ее главное преимущество — копирование только измененных частей файлов, что делает ее очень быстрой.
Синтаксис: rsync [ОПЦИИ] ИСТОЧНИК [ИСТОЧНИК...] НАЗНАЧЕНИЕ
Пример: rsync -avz ~/Documents user@remote_host:/backup/
Примечание: Обратите внимание на слеш (/) в конце пути к источнику. Он влияет на то, будет скопирована сама папка или ее содержимое.

===
Название: watch
Описание: Запускает указанную команду и показывает ее вывод в полноэкранном режиме, позволяя наблюдать за изменениями в реальном времени.
Синтаксис: watch [ОПЦИИ] КОМАНДА
Пример: watch -d -n 0.01 nvidia-smi
Примечание: По умолчанию интервал обновления — 2 секунды. Изменяется флагом -n.

===
Название: dd
Описание: Низкоуровневая утилита для конвертации и копирования файлов, часто используется для работы с блочными устройствами (создание загрузочных флешек, клонирование дисков, создание файлов-образов).
Синтаксис: dd if=ИСТОЧНИК of=НАЗНАЧЕНИЕ [ОПЦИИ]
Пример: dd if=/dev/sr0 of=disk_image.iso (создание ISO-образа CD/DVD)
Примечание: Крайне опасная команда при неправильном использовании! Неправильное указание of= (например, на свой системный диск) приведет к безвозвратной потере данных. Всегда перепроверяйте аргументы.

===
Название: journalctl
Описание: Утилита для просмотра логов, собранных системным демоном journald (используется в современных дистрибутивах на базе systemd).
Синтаксис: journalctl [ОПЦИИ]
Пример: journalctl -u nginx.service (логи только для службы nginx)
Примечание: Незаменимый инструмент для отладки проблем со системными службами.

===
Название: tmux (Terminal Multiplexer)
Описание: Позволяет создавать несколько сессий, окон и панелей (разделений) внутри одного терминала. Сессия tmux продолжит работать, даже если вы закрыли терминал или потеряли SSH-соединение.
Синтаксис: tmux [КОМАНДА] [ОПЦИИ]
Пример:
    tmux new -s mysession (создать новую сессию с именем 'mysession')
    Ctrl+b % (разделить окно по вертикали)
    Ctrl+b " (разделить окно по горизонтали)
    tmux attach -t mysession (переподключиться к сессии 'mysession')
Примечание: Имеет собственную систему управления на основе префикса (по умолчанию Ctrl+b).

===
Название: curl
Описание: Утилита командной строки для передачи данных с или на сервер с использованием различных сетевых протоколов (HTTP, HTTPS, FTP, SCP и многих других). Незаменима для тестирования API, загрузки файлов и автоматизации веб-запросов.
Синтаксис: curl [ОПЦИИ] [URL...]
Пример: curl https://google.com
Примечание: Один из самых мощных и часто используемых инструментов в арсенале разработчика и системного администратора.

===
Команда 13: sshfs (SSH Filesystem)

Название: sshfs
Описание: Позволяет монтировать удаленную файловую систему через SSH-соединение с помощью FUSE. После монтирования вы можете работать с файлами на удаленном сервере так, как если бы они были на вашей локальной машине.
Синтаксис: sshfs [USER@]HOST:[КАТАЛОГ] ТОЧКА_МОНТИРОВАНИЯ [ОПЦИИ]
Пример: sshfs user@192.168.1.100:/home/user/ ~/remote_server/
Для размонтирования: fusermount -u ~/remote_server
Примечание: Требует установки пакета sshfs. После монтирования все операции с файлами идут по сети, поэтому производительность зависит от скорости соединения.

===
Название: column
Описание: Форматирует вывод текста в несколько колонок с выравниванием. Особенно полезна для красивого отображения табличных данных, например, вывода скриптов.
Синтаксис: column [ОПЦИИ] [ФАЙЛ]...
Пример: printf "NAME AGE GENDER\nAlice 25 F\nBob 30 M\nCharlie 35 M" | column -t
Результат:
text

NAME     AGE  GENDER
Alice    25   F
Bob      30   M
Charlie  35   M

Примечание: Ключ -t определяет количество колонок автоматически и создает таблицу. Простая, но очень эффективная команда для оформления вывода.
